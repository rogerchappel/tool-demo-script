# Fixture Demo Promotion Brief

## Positioning

tool-demo-script helps CLI maintainers turn package metadata into a runnable demo
script, then checks the generated commands against the repo.

## Hooks

- "A CLI demo should be more than copy; it should still run."
- "Generate a Markdown walkthrough and narration metadata from a package.json."
- "Keep demo scripts honest with a verification pass over safe commands."

## Video Outline

1. Show the fixture CLI package metadata.
2. Run `bash demo/run-fixture-demo.sh`.
3. Inspect the generated Markdown demo.
4. Open `narration.json` to show reusable video prep metadata.
5. Explain the safe command allowlist for verification.
